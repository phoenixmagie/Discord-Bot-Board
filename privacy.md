# Privacy Policy / Datenschutzerklärung

### English
1. **Data Processing:** To provide its services, the Bot processes Server IDs, User IDs, and Channel IDs.
2. **Hosting & Control:** This Bot is hosted and managed using **BotGhost**. The developer of this Bot does not have direct access to the underlying database and cannot manually delete individual data entries. 
3. **Data Deletion:** All data management, including storage and deletion, is handled by the BotGhost infrastructure. To request a complete deletion of data, users may need to remove the Bot from their server or contact BotGhost support directly.
4. **Usage:** Data is exclusively used for core features and will never be sold by the developer.

### Deutsch
1. **Datenverarbeitung:** Zur Bereitstellung der Funktionen verarbeitet der Bot Server-IDs, User-IDs und Kanal-IDs.
2. **Hosting & Kontrolle:** Dieser Bot wird über **BotGhost** gehostet und verwaltet. Der Entwickler dieses Bots hat keinen direkten Zugriff auf die zugrunde liegende Datenbank und kann keine manuellen Löschungen von Datensätzen vornehmen.
3. **Datenlöschung:** Die gesamte Datenverwaltung, einschließlich Speicherung und Löschung, erfolgt über die Infrastruktur von BotGhost. Um eine vollständige Löschung der Daten zu bewirken, müssen Nutzer den Bot möglicherweise vom Server entfernen oder sich direkt an den Support von BotGhost wenden.
4. **Nutzung:** Daten werden ausschließlich für Kernfunktionen verwendet und niemals durch den Entwickler an Dritte verkauft.
